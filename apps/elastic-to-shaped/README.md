This is a demo app showing how easy it is to convert Elastic DSL code to a Shaped Engine. 

## Local development

Follow these steps if you want to develop on this app locally. 

### LLM environment setup

This app uses Claude to provide code suggestions. Add your API key to an .env file as `ANTHROPIC_API_KEY`

### Run development server

Run the development server:

```bash
npm run dev
```
